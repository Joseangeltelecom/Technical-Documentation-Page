Introduction

Oracle Primavera P6 es uno de los softwares de gestión de proyectos más útiles y eficaces disponibles en el mercado. Está diseñado para planificar y gestionar todo tipo de proyectos complejos con satisfacer los requisitos de cada miembro del equipo.

Este software proporciona todas las herramientas y características necesarias que ayuda a gestionar todo el ciclo de vida de sus proyectos. La gestión de riesgos, el análisis de rutas críticas, los recursos y la gestión de costes son algunas de las características básicas de este software. Con la ayuda de ella, las organizaciones pueden gestionar con éxito sus proyectos en las condiciones económicas inciertas de hoy en día

¿Que es Oracle Primavera P6 EPPM?

Es una buena idea entender la diferencia entre P6 PPM y P6 EPPM antes de comprar uno de ellos. Ambos son adecuados para proyectos grandes y desafiantes. Sin embargo, P6 PPM es un software basado en Windows que se conecta a una base de datos basada en servidor o local. Aunque muchas organizaciones prefieren utilizar P6 PPM como software independiente, se puede utilizar como una herramienta independiente o una herramienta empresarial.

P6 EPPM incluye muchas características útiles que P6 PPM no tiene. Una de ellas es la herramienta de Gestión de Metodología que implica plantillas de proyecto para diferentes tipos de proyectos. A partir de la versión 8, la gestión de metodología se elimina de P6 PPM y ahora sólo está disponible en P6 EPPM.

Otra diferencia importante es que la velocidad de procesamiento de P6 PPM es mayor que Primavera P6 EPPM. Porque es un software independiente y esta capacidad permite realizar cambios más rápido. Por otro lado, P6 EPPM requiere una conexión a Internet de alta calidad y si su velocidad de conexión es baja, no puede utilizar el software de manera eficiente.

La función de generación de informes de P6 EPPM es más avanzada que P6 PPM. Puede crear más informes visuales, gráficos, tablas en P6 EPPM. Esto es útil para demostrar el estado del proyecto en cualquier nivel, desde el nivel ejecutivo hasta los niveles más bajos.

Caracteristicas Principales

Primavera P6 ofrece soluciones de gestión de proyectos de alto nivel para su negocio. Muchas personas piensan que Primavera P6 se crea sólo para fines de programación. No es sólo una herramienta para crear diagramas de Gantt. Proporciona muchas soluciones como gestión de costes, gestión del valor ganado, gestión de riesgos, nivelación de recursos, gestión de contratos e informes.

Revisemos las principales características del Primavera P6:

  • Programación: Primavera P6 implica soluciones avanzadas de programación. Las actividades se pueden crear en los niveles PEP (Estructura de desglose de trabajo), se pueden establecer relaciones entre las actividades y la ruta crítica de la programación se puede calcular mediante la ayuda de la función de programación.
  • Gestión de Recursos y Control de Costos: Los recursos, los precios unitarios y los gastos se pueden asignar a las actividades. El presupuesto del proyecto se puede realizar un seguimiento con la ayuda de esta característica. Los equipos del proyecto pueden comparar fácilmente el estado planificado y real de los costos y analizar el rendimiento del proyecto.
  • Gestión de Riesgos: Es posible identificar y priorizar riesgos con la ayuda de esta función. Los riesgos se pueden asignar a actividades y se puede realizar análisis de riesgos para cada actividad y proyecto.
  • Gestión de contratos: La función de gestión de contratos Primavera P6 proporciona soluciones avanzadas de control de documentos. Es posible realizar un seguimiento del proceso de solicitudes de información, cambios, problemas, envío y aprobaciones y su impacto en la programación con esta característica.
  • Informes: Primavera P6 cuenta con una función avanzada de informes que permite utilizar los informes existentes en la base de datos o crear informes personalizados para necesidades especiales.

  • Ayuda a gestionar los riesgos del proyecto

The most common Earned Value formulas and calculations
  • Actual Cost (ACWP): ACWP = Actual Labour Cost + Actual Non-Labour Cost + Actual Material Cost + Actual Expense Cost.
  • Cost Performance Index (CPI): CPI = EV / Actual Cost.
  • Cost Variance (CV): CV = EV – Actual Cost.
  • Earned Value Cost (BCWP or EV): EV = BAC * Performance % Complete.
  • Estimate At Completion (EAC): EAC = Actual Cost + ETC.
  • Estimate to Complete (ETC): ETC = PF * (BAC – EV).
  • Planned Value Cost (BCWS or PV): PV = BAC * Schedule % Complete.
  • Schedule Performance Index (SPI): SPI = EV / PV.
